# yandex-predictor-kt

Asynchronous Yandex.Predictor API wrapper for Kotlin/JVM.

### Installation

```kotlin
repositories {
    maven(url = "https://jitpack.io")
}

dependencies {
    implementation("dev.d1s:yandex-predictor-kt:$yandexPredictorVersion")
}
```
